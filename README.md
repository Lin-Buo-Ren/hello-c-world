# HelloCworld
這是個最簡單的「Ｃ程式語言世界哈囉！」程式

## 如何建構專案
在命令列介面中切換當前工作目錄(current working directory)到「HelloCworld」專案來源程式碼根目錄下，然後執行下列命令：
```
$ bash build.bash.sh
```

## 如何執行建構出來的程式
在命令列介面中切換當前工作目錄(current working directory)到「HelloCworld」專案來源程式碼根目錄下，然後執行下列命令：
```
$ ./HelloCworld.exe
```
